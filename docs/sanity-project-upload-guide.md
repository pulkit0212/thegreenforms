# 📋 Sanity CMS — Project Upload Guide

## How to Access the CMS

1. Open your browser and go to: **https://thegreenforms.com/studio**
2. Log in with your Sanity account (email you were invited with)
3. You'll see the Studio dashboard with **"Project"** in the left sidebar

---

## How to Create a New Project

1. Click **"Project"** in the left sidebar
2. Click the **＋ (Create)** button at the top
3. Fill in the fields:

| Field | What to Enter |
|---|---|
| **Project Title** | Name of the project (e.g. "The Royal Villa") |
| **URL Slug** | Click **"Generate"** — it auto-creates from the title |
| **Category** | Select one: Residential, Commercial, Office, or Retail |
| **City** | Select: Jaipur, Gurgaon, or Other |
| **Year** | Year completed (e.g. "2024") |
| **Featured Project** | Toggle ON if you want it on the homepage |
| **Short Description** | 1–2 sentence summary for the project card |
| **Full Description** | Detailed write-up for the project detail page |

---

## How to Upload the Cover Image

1. Scroll to **"Cover Image"**
2. Click **"Upload"** or drag-and-drop your photo
3. This is the main image shown on project cards and listings
4. Use a high-quality vertical or square photo
5. (Optional) Add **Alt Text** — a short description like "Living room interior view"

---

## How to Upload Gallery Images

1. Scroll to **"Gallery Images"**
2. Click **"Add item"** to add one image at a time, or drag multiple files at once
3. You can add **as many images as you like**
4. **Drag to reorder** — the order you set is the order they appear on the website
5. (Optional) Add **Alt Text** for each image

---

## How to Add Tags

1. Scroll to **"Tags"**
2. Type a keyword and press **Enter** — e.g. "Modern", "Luxury", "Minimal"
3. Tags help visitors find similar projects

---

## How to Set Sort Order

- Use the **"Sort Order"** field to control the display order
- **Lower numbers appear first** (e.g. 1 shows before 10)
- Leave empty for default sorting

---

## How to Publish

1. After filling in all fields, click the green **"Publish"** button (bottom-right)
2. The project will appear on the website within a few minutes
3. To edit later, click any project from the list and make changes, then **Publish** again

---

## 📸 Image Guidelines

### Recommended Image Sizes

| Image Type | Recommended Size | Format |
|---|---|---|
| Cover Image | 1200 × 1600 px (3:4 ratio) | JPG or WebP |
| Gallery Images | 1600 × 1200 px (landscape) | JPG or WebP |

### Best Practices

- **File size**: Keep each image under **500 KB** for fast loading
- Compress images before uploading using [tinypng.com](https://tinypng.com) (free)
- **File naming**: Use descriptive names like `living-room-view.jpg` (not `IMG_4521.jpg`)
- **How many images**: 5–15 gallery images per project is ideal
- **Quality**: Use well-lit, professional photos — they represent your brand

### Quick Compression Workflow

1. Go to [tinypng.com](https://tinypng.com)
2. Drag your photos (up to 20 at a time)
3. Download the compressed versions
4. Upload the compressed versions to Sanity
